Model Architecture

Documentation for the neural network architecture used in the neural-lam weather prediction system.

Graph Neural Network Architecture

The neural-lam project implements Graph Neural Networks (GNNs) for weather forecasting. The core models operate on a hierarchical mesh structure:

        L1 <--> L2 <--> L3
    end

    subgraph Decoder ["🔄 Mesh-to-Grid Decoder"]

    end

    GRID --> G2M --> L1
    L1 --> M2G --> PRED

Message Passing Scheme

Each GNN layer performs message passing as follows:

  1. Edge Update: Compute edge features from source/target node features

  2. Aggregation: Aggregate incoming edge messages per node (sum)

  3. Node Update: Update node features using aggregated messages + self-features

# Pseudo-code for a single GNN layer
def forward(self, x, edge_index, edge_attr):
    """
    Parameters
    ----------
    x : torch.Tensor
        Node features of shape ``(N, d_node)``.
    edge_index : torch.Tensor
        Edge connectivity of shape ``(2, E)``.
    edge_attr : torch.Tensor
        Edge features of shape ``(E, d_edge)``.

    Returns
    -------
    torch.Tensor
        Updated node features of shape ``(N, d_node)``.
    """
    # Edge update
    edge_msg = self.edge_mlp(cat([x[src], x[dst], edge_attr], dim=-1))
    # Aggregate
    agg = scatter_sum(edge_msg, dst, dim=0, dim_size=x.size(0))
    # Node update
    x_out = self.node_mlp(cat([x, agg], dim=-1))
    return x_out
