Register a new Organization.

This endpoint initializes a new tenant space for enterprise customers.

Args:

name (str): The legal name of the organization. account_type (AccountType): The subscription tier (Basic, Premium, Enterprise). emails (List[str]): List of administrative contact email addresses.

Returns:

Organization: The newly created organization record.
