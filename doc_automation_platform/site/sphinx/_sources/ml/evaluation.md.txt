# Evaluation Metrics

> Structure and documentation of the evaluation metrics used to assess
> weather prediction model performance in neural-lam.

## Metric Definitions

### Root Mean Square Error (RMSE)

$$
\text{RMSE}(v, t) = \sqrt{\frac{1}{N} \sum_{i=1}^{N} w_i \left( \hat{y}_{i,v,t} - y_{i,v,t} \right)^2}
$$

where:
- $v$ = variable index, $t$ = lead time
- $N$ = number of grid points
- $w_i$ = area weight for grid point $i$ (accounts for latitude)
- $\hat{y}$ = prediction, $y$ = ground truth

| Aspect | Detail |
|--------|--------|
| **Range** | $[0, \infty)$ — lower is better |
| **Units** | Same as the variable (K, m/s, Pa, etc.) |
| **Weighting** | Area-weighted (cosine of latitude) |

### Mean Absolute Error (MAE)

$$
\text{MAE}(v, t) = \frac{1}{N} \sum_{i=1}^{N} w_i \left| \hat{y}_{i,v,t} - y_{i,v,t} \right|
$$

| Aspect | Detail |
|--------|--------|
| **Range** | $[0, \infty)$ — lower is better |
| **Units** | Same as the variable |
| **Robustness** | More robust to outliers than RMSE |

### Anomaly Correlation Coefficient (ACC)

$$
\text{ACC}(v, t) = \frac{\sum_i w_i \, \hat{a}_{i,v,t} \, a_{i,v,t}}{\sqrt{\sum_i w_i \, \hat{a}_{i,v,t}^2} \sqrt{\sum_i w_i \, a_{i,v,t}^2}}
$$

where anomalies are computed relative to climatology:
$a = y - \bar{y}_\text{clim}$, $\hat{a} = \hat{y} - \bar{y}_\text{clim}$

| Aspect | Detail |
|--------|--------|
| **Range** | $[-1, 1]$ — higher is better |
| **Threshold** | ACC > 0.6 is considered "useful" forecast skill |
| **Reference** | Climatological mean |

## Evaluation Schema

```{mermaid}
classDiagram
    class EvaluationResult {
        +str model_name
        +str dataset_name
        +datetime timestamp
        +dict~str,MetricResult~ metrics
        +int num_samples
        +list~int~ lead_times
    }

    class MetricResult {
        +str metric_name
        +dict~str,float~ per_variable
        +dict~int,float~ per_lead_time
        +dict~str,dict~ per_level
        +float global_mean
    }

    class VariableScore {
        +str variable_name
        +str units
        +float score
        +float std_error
    }

    EvaluationResult --> MetricResult : contains
    MetricResult --> VariableScore : per variable
```

## Score Degradation by Lead Time

```{mermaid}
graph LR
    T6["6h\n⭐⭐⭐⭐⭐"] --> T12["12h\n⭐⭐⭐⭐"]
    T12 --> T24["24h\n⭐⭐⭐"]
    T24 --> T48["48h\n⭐⭐"]
    T48 --> T72["72h\n⭐"]

    style T6 fill:#c8e6c9,stroke:#2e7d32
    style T12 fill:#dcedc8,stroke:#558b2f
    style T24 fill:#fff9c4,stroke:#f9a825
    style T48 fill:#ffe0b2,stroke:#ef6c00
    style T72 fill:#ffccbc,stroke:#bf360c
```

Forecast skill degrades with lead time. The standard operational threshold is
ACC > 0.6 for "useful" skill. For neural-lam models, this is typically reached
around 5–7 days for 500 hPa geopotential.
