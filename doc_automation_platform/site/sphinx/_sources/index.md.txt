# AutoCodeDoc Platform - Sphinx Documentation

Welcome to the Sphinx version of the AutoCodeDoc Platform documentation. 

This documentation is built alongside MkDocs. By supporting both, we ensure better alignment with the GSoC proposal and provide viewers the flexibility to choose their preferred documentation format.
